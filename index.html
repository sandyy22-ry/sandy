Materi Basis Data

Basis data adalah kumpulan data yang terorganisir dan dapat diakses dengan mudah secara elektronik. Sistem Manajemen Basis Data (DBMS) seperti MySQL, PostgreSQL, dan SQLite membantu kita mengelola data secara efisien.

Konsep Dasar:

  • Database â€” tempat penyimpanan data.
  • Tabel â€” struktur yang menyimpan data dalam baris dan kolom.
  • Primary Key â€” kolom unik untuk setiap baris data.
  • Relasi â€” hubungan antar tabel (One-to-One, One-to-Many, Many-to-Many).

Contoh Query SQL:

CREATE TABLE Mahasiswa (
  NIM CHAR(10) PRIMARY KEY,
  Nama VARCHAR(50),
  Jurusan VARCHAR(30)
);

INSERT INTO Mahasiswa VALUES ('2401010001', 'Sandy Tirta', 'Informatika');

Tugas Basis Data

  1. Buat database kampus di MySQL.
  2. Buat tabel mahasiswa dengan kolom NIM, Nama, Jurusan.
  3. Masukkan minimal 5 data mahasiswa.
  4. Tampilkan seluruh data mahasiswa dengan perintah SELECT *.
  5. Ubah nama salah satu mahasiswa menggunakan UPDATE.
  6. Hapus satu data dengan DELETE.
